# 🛡️ Análise de Cibersegurança e Riscos Financeiros com NotebookLM

Este repositório contém o desafio de laboratório realizado para a **DIO (Digital Innovation One)**, focado em utilizar o **NotebookLM** como ferramenta de aprendizagem ativa e curadoria de dados.

## 🎯 Objetivo
O objetivo foi analisar o impacto financeiro de incidentes de cibersegurança e fraudes digitais, utilizando relatórios de alta autoridade como fontes primárias para a IA.

## 🛠️ Tecnologias e Ferramentas
* **NotebookLM:** Para síntese de dados e criação de miniguias.
* **Engenharia de Prompt:** Aplicação de técnicas *Few-shot* e *Chain-of-Thought*.
* **Fontes Analisadas:** Relatórios da IBM (Cost of a Data Breach), FEBRABAN e Banco Central.

## 📋 Conteúdo do Projeto
* **Glossário Técnico:** Termos essenciais que conectam Finanças e Segurança.
* **Guia de Resposta a Incidentes:** Passo a passo para mitigação de prejuízos financeiros.
* **Perguntas Estratégicas:** Validação de pontos críticos sobre vazamento de dados.
